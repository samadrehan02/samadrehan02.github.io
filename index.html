Flagship Projects

AI Medical Scribe

Real-time medical transcription and clinical note generation from noisy doctor–patient conversations in outpatient settings.

System Overview:
Live audio stream → Hindi/Hinglish ASR → entity extraction → constrained LLM generation → structured OPD note + export

  • Chose a lightweight ASR pipeline optimized for real-time latency and conversational Hindi/Hinglish instead of maximizing offline WER scores
  • Performed clinical entity extraction before LLM generation to reduce hallucinations and enforce medical safety boundaries
  • Used instruction-tuned LLMs with strict output schemas rather than free-form generation to maintain consistency and auditability
  • Designed session-scoped workflows with speaker handling to support continuous, real-time clinical usage

Constraints: real-time audio, noisy environments, mixed-language speech, clinical safety, low tolerance for hallucinations
