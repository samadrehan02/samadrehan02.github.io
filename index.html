Flagship Projects

AI Medical Scribe

Real-time medical transcription and clinical note generation from noisy doctor–patient conversations in outpatient settings.

System Overview:
Live audio stream → Hindi/Hinglish ASR → entity extraction → constrained LLM generation → structured OPD note + export

  • Chose a lightweight ASR pipeline optimized for real-time latency and conversational Hindi/Hinglish instead of maximizing offline WER scores
  • Performed clinical entity extraction before LLM generation to reduce hallucinations and enforce medical safety boundaries
  • Used instruction-tuned LLMs with strict output schemas rather than free-form generation to maintain consistency and auditability
  • Designed session-scoped workflows with speaker handling to support continuous, real-time clinical usage

Constraints: real-time audio, noisy environments, mixed-language speech, clinical safety, low tolerance for hallucinations

Minutes of Meeting AI

Automated meeting transcription and summarization system focused on extracting decisions, action items, and accountability from multi-speaker conversations.

System Overview:
Raw transcript → speaker normalization → structured information extraction → constrained LLM summarization → actionable meeting notes

  • Prioritized structured extraction of decisions, action items, and owners instead of relying on generic abstractive summaries
  • Designed prompts and post-processing to handle noisy, multi-speaker transcripts with interruptions and partial sentences
  • Used schema-constrained LLM outputs to ensure consistency across meetings and enable downstream automation
  • Optimized for usefulness in follow-ups and accountability rather than narrative completeness

Constraints: noisy transcripts, overlapping speakers, implicit decisions, need for consistent structure across meetings

TCN Stock Prediction

Time-series stock forecasting system using a hybrid Temporal Convolutional Network and Transformer architecture, designed for reproducibility and stable inference.

System Overview:
Historical price data → feature normalization → TCN for local temporal patterns → Transformer for long-range dependencies → forecasting output

  • Combined TCNs for efficient modeling of short-term temporal structure with Transformers for capturing longer-range dependencies
  • Prioritized reproducibility and stable inference behavior over one-off benchmark optimization
  • Built an end-to-end pipeline with clear data splits to reduce leakage and improve evaluation reliability
  • Focused on inference efficiency to support repeated forecasting rather than single-run experimentation

Constraints: non-stationary time-series data, risk of data leakage, need for repeatable evaluation, inference efficiency
